Web Skills

JavaScript is the most popular scripting language used, and there's a good reason why. It allows for developers to add interactivity with ther web applications that normal HTML and CSS couldn't do. Sometimes it can be fun to work with, but other times it can be difficult and challenging. But with my experience, I've always enjoyed applying it to my work.

JQuery is an extensive JavaScript library that adds a lot of usability to your websites. It has the motto "Write less, do more" which is exactly what you do with JQuery. It simplifies JavaScript and does a lot more. JQuery is great for animations too, and even has multiple plugins to help as well. One of its core features is how it traverses the DOM on your website, allowing for simple selection and editing.

HTML is the foundation of developing websites. Without HTML, there's nothing to reference to for styling or functionality. Good HTML semantics improve overall design, functionality, and speed of your website. Using correct semantics also helps other developers understand your code if you're working on a team project, it also helps you understand what you're doing.

CSS is where all the fun is at in web design. Not only is it used for styling, it's also great for animations. There are unlimited possibilities with CSS animations. From spinning objects to walking sprites, CSS can do it all. Along with animation, CSS is the core styling language for websites. You can't have a website without CSS. They go hand in hand. And only the best of the best know how to make cool websites with it.
